Categories of Mobile Applications

Native Applications

These are applications which can be accessed through the mobile device's app store to be downloaded on to the device. This serves as a way which users can easily gain access to these applications as well as these apps can tap into the wider functionality of the device whiich includes the camera, microphone, compass, accelerometer and swipe gestures amongst other functionalities that the mobile device may come with.

Hybrid Applications

These are apps that are built with a combination of web technologies like HTML, CSS, and JavaScript. They are fast and easy to develop and hence are much cheaper compared to applications such as Native apps.

Web Applications

These Apps are apps that can be accessed on your web browswer without the need of being able to access the app version. These usually require minimum of device performance and use a bit of device memory and provides most of the Functionalities that the app verion may provide.
